# 10｜Decision Log 模板

新增决定时追加，不覆盖历史。

```markdown
## DEC-YYYYMMDD-XX：<标题>

- 时间：
- 提出者：
- 状态：Proposed / Accepted / Rejected / Superseded
- 影响范围：
- 截止时间：

### 背景
...

### 选项
1. ...
2. ...

### 决定
...

### 原因
...

### 契约影响
- OpenAPI：
- Schema：
- Fixture：
- Swift：
- TypeScript：

### 风险与降级
...

### Owner
...
```

必须记录：

- Photon line/渠道最终能力。
- DeviceActivity 是否进入真实主线。
- Gmail scope 与 Demo 授权结果。
- Blue Box 内容来源。
- Must 功能任何变更。
- Contract v1 冻结与后续升级。
